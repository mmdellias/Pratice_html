Página de Áudio

Essa é uma gravação de teste só para mostrar como funciona a tag de áudio aqui no HTML. Que é muito utilizado para sites onde você terá a leitura do texto através do áudio. Sites que tem podcast, sites que tem conteúdo e uma área de áudio para que possa ter a leitura, utiliza esse recurso. É bem simples de se usar a tag de áudio, nessa aula vocês estão vendo como fazer isso.
